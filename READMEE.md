## Getting Started

### Requirements

- Xcode 26.3 or later
- iOS 26.2+ device or simulator

### Running the App

1. Clone the repo
   ```bash
   git clone https://github.com/your-username/OnePiece.git
   ```
2. Open `OnePiece.xcodeproj` in Xcode
3. Select a simulator or connected device
4. Press **⌘ R** to build and run
